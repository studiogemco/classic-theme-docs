Install the Theme

  1. In your WordPress Admin sidebar, navigate to Appearance > Themes.
  2. Click Add New Theme.
  3. On the next page, click Upload Theme.
  4. Click Choose File, locate your theme’s .zip file, and select it.
  5. Click Install Now, then on the next page, click Activate.

Install Recommended Plugins

  1. You should see a message box that says “The following recommended plugins are currently inactive:”. In this box, click Begin activating plugins.
  2. On the next page, select/check all plugins. In the Bulk Actions dropdown, select Install, then click Apply.
  3. On the next page, click Return to the Dashboard, or simply click Dashboard in your WordPress sidebar.

Import Demo Content

Please note: This theme has been retired, and as a result, the demo content is no longer available. While the content will still be imported, media files such as images will not be included.

  1. In your WordPress Admin sidebar, navigate to Appearance > Import Demo Data.
  2. Click Import Demo Data.
  3. On the next page, click Continue & Import.
  4. Once the process is complete, you’ll see the “Import Complete” message.

[ERROR] Failed to import … (Media)

Since this theme has been retired and the demo is no longer live, you will encounter [ERROR] Failed to import … (Media) for each image from the original demo. The missing images are the only elements not included in the imported content. You can remove the sample posts and replace any broken images on your pages as needed.

Delete Sample Posts

The sample posts were provided only to demonstrate how the theme looks with content. Instead of editing them, they should be deleted.

  1. In your WordPress Admin sidebar, navigate to Posts > All Posts.
  2. In the All Categories dropdown, choose Sample Category and click Filter.
  3. Check the top checkbox to select all sample posts.
  4. From the Bulk Actions dropdown, choose Move to Trash, then click Apply.
